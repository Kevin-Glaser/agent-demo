# 代码审查技能

## 使用时机

当用户执行下列操作时,应使用此技能:
- 代码审查
- 请求审查代码
- 需要改进建议

## 审查流程

### 1. 代码审查
- 检查代码规范
- 验证代码结构
- 检查重复代码

### 2. 最佳实践
- 确保模式使用正确
- 检查安全性
- 验证性能影响
